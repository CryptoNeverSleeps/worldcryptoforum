# World Crypto Forum Coin

## WCF Specifications

* **Ticker : WCF**
* **Type : POS/MN(Masternode)**
* **Algorithm : X11**
* **Block Time : 1min**
* **Block reward 0.5 - 14**
* **MN Reward : 90%**
* **Pos Reward : 10%**
* **Min Stake time : 1hours**
* **Total supply : 30,000,000**
* **Premine : 300,000(Only 1%)**
